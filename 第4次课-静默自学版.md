# 第 4 次课｜深入项目：代码全景与首次完整实验（静默自学版）

> 课堂无口述、无演示。教师仅提供计时与文字提示；学生按步骤独立完成。

## 目标
- 跑通项目基线并理解核心模块。
- 形成首次完整实验记录与指标结果。

## 课前准备
- 已确定项目方向并具备代码仓库地址。
- 具备基本 Python 环境（建议使用虚拟环境）。

## 国内网络提示
- GitHub 访问慢时，优先使用 Gitee 镜像或课程提供的代码包。
- 权重或数据下载慢时，先询问是否有离线包。

## 课堂流程（静默）
- 阅读项目结构与运行说明 15 min。
- 克隆项目并配置依赖 35 min。
- 运行基线实验并记录指标 50 min。
- 整理日志与问题清单 20 min。

## 操作步骤（照做即可）
1. 克隆/解压项目到本地目录（如 `~/research-lab/project`）。
2. 创建虚拟环境并安装依赖，若失败记录具体报错。
3. 阅读 `README` 与配置文件，定位入口脚本与数据路径。
4. 运行最小基线实验（先用最小参数/小数据）。
5. 保存日志、截图与关键指标（如准确率、损失）。

## 产出与提交
- 基线结果表或日志截图。
- 项目结构笔记（模块与入口说明）。

## 自检清单
- 项目能在本机启动并完成一次训练/推理。
- 关键配置文件能说明其作用。
- 已保存基线指标。

## 常见问题
- 依赖冲突：优先使用项目推荐版本；必要时新建虚拟环境。
- 缺少数据：联系课程资源或使用官方样例数据。
- 运行太慢：先降小 batch/epoch，确保流程跑通。
