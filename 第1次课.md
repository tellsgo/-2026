# 第 1 次课｜从零开始：WSL2、Ubuntu 与第一个贪吃蛇（静默自学版）

> 课堂无口述、无演示。教师仅提供计时与文字提示；学生按步骤独立完成。

## 目标
- 在 Windows 上完成 WSL2 + Ubuntu 的可用环境。
- 能在 Ubuntu 中运行一个最小可用的贪吃蛇程序。
- 形成第一份实验记录（含截图与关键命令）。

## 课前准备（学生自查）
- Windows 10/11，开启虚拟化（BIOS/任务管理器中显示“已启用虚拟化”）。
- 稳定网络；若学校网络受限，准备手机热点备用。
- 预留 C 盘 20GB 以上空间。

## 国内网络提示
- Microsoft Store 访问不稳定时，可用系统命令安装 WSL2（见指导书）。
- 包下载缓慢时，优先使用校园网；必要时切换到国内镜像源。

## 课堂流程（静默）
- 读指导书相关章节 10 min。
- 安装 WSL2 与 Ubuntu 50 min。
- 完成贪吃蛇任务并运行 50 min。
- 整理截图与记录 20 min。
- 提交与自检 10 min。

## 操作步骤（照做即可）
1. 按指导书步骤启用 WSL2 功能并安装 Ubuntu。
2. 首次进入 Ubuntu 后完成基础更新与常用工具安装（git、python、pip）。
3. 创建课程工作目录（例如 `~/research-lab/lesson1`）。
4. 打开 Cursor（或其他编辑器）连接 WSL2，建立新项目。
5. 使用提示词生成贪吃蛇最小可运行代码（控制台或 pygame 皆可）。
6. 运行程序并截图：
   - 若 GUI 显示异常，先确认 WSLg 是否启用，再检查依赖。
7. 将关键命令、截图与问题记录整理到实验记录模板。

## 产出与提交
- 贪吃蛇运行截图或短录屏。
- 实验记录（包含环境版本、关键命令、问题与解决）。

## 自检清单
- 能正常进入 Ubuntu 终端。
- `python --version` 可用，`pip` 可用。
- 贪吃蛇程序能启动并响应键盘。

## 常见问题
- 安装失败：优先检查系统版本与虚拟化设置。
- 下载慢：切换网络或使用国内镜像。
- 图形界面不可用：确认 WSLg 支持与 Windows 更新状态。
