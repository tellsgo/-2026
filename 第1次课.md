  Step 1 → 环境安装

  Step 2 → 基础配置
  首次进入 Ubuntu 后执行：
   # 系统更新
   sudo apt update && sudo apt upgrade -y

   # 安装基础工具
   sudo apt install git python3 python3-pip -y

   # 配置 iFlow
   # （按指导书完成配置）

  Step 3 → 项目初始化
   # 创建工作目录
   mkdir -p ~/research-lab/lesson1
   cd ~/research-lab/lesson1

  Step 4 → 编辑器连接
  打开 Cursor 或其他编辑器，连接至 WSL2 并创建新项目

  Step 5 → 代码生成
  使用提示词生成贪吃蛇最小可运行代码（控制台或 pygame 版本均可）

  ⚠ 常见问题排查

   问题 1：安装失败
   ├─ 原因：系统版本不兼容或虚拟化未开启
   └─ 解决：检查系统版本并确认虚拟化设置

   问题 2：下载速度慢
   ├─ 原因：网络连接不稳定
   └─ 解决：切换网络或使用国内镜像源

   问题 3：图形界面不可用
   ├─ 原因：WSLg 未启用或 Windows 未更新
   └─ 解决：确认 WSLg 支持及 Windows 更新状态

   问题 4：磁盘附加错误
   ├─ 症状："无法将磁盘 C:\Program Files\WSL\system.vhd 附加到 WSL2"
   ├─ 原因：VHDX 文件在重启后被系统自动删除
   └─ 解决：清理残留文件后重新运行安装程序
