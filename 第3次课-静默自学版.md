# 第 3 次课｜读论文不用读论文：Vibe Researching（静默自学版）

> 课堂无口述、无演示。教师仅提供计时与文字提示；学生按步骤独立完成。

## 目标
- 选择一个课程方向并完成快速文献梳理。
- 形成 1 个明确研究问题与基线理解。
- 输出一页研究地图或结构化摘要。

## 课前准备
- 从课程候选项目中选定方向（如语义通信、信道估计、编码等）。
- 备好阅读与整理工具（笔记软件或 Markdown）。

## 国内网络提示
- Scholar 访问不稳定时，优先使用 arXiv、ACL Anthology、IEEE Xplore 校园网入口或课程提供的论文包。
- GitHub 不稳定时，先找 Gitee 镜像或项目 Release 包。

## 课堂流程（静默）
- 阅读 Vibe Researching 流程卡 10 min。
- 收集 3–5 篇关键论文/开源项目 40 min。
- 生成研究地图与问题列表 40 min。
- 完成问题陈述与基线描述 30 min。

## 操作步骤（照做即可）
1. 写下“我想解决的问题是什么？”用一句话描述。
2. 搜索 3–5 篇代表性论文或开源项目，记录标题与链接。
3. 用提示词生成研究地图（背景 → 方法 → 数据/指标 → 结论）。
4. 选择一个可验证的研究问题，写出基线做法与不足。
5. 输出 1 页结构化摘要（可用 Markdown）。

## 产出与提交
- 一页研究地图/结构化摘要。
- 参考文献清单（含链接）。

## 自检清单
- 研究问题清晰且可实验验证。
- 已描述至少一个基线方法。
- 参考文献数量 ≥ 3。

## 常见问题
- 找不到论文：缩小范围，从综述或代表性会议开始。
- 阅读困难：先看摘要、方法图与实验表格。
- 选题过大：收敛到一个可验证指标或单一改进点。
